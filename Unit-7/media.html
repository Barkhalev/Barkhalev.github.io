audio - для вывода аудиофайлов на страницу
video - для вывода видеофайлов на страницу
source - для указания разных форматов медиафайлов на страницу

Audio

  • src - путь до файла который будет играть
  • control - показывать элементы управления
  • autoplay - сразу начать проигрывать файл
  • muted - включить беззвучный режим
  • loop - бесконечно проигрывать звуковой файл
  • preload - как браузер будет загружать файл
  • none - не загрузит пока не нажмешь воспроизведение
  • metadata
